=================

Language: Swift 5.9
UI Framework: SwiftUI (declarative)
Persistence: SwiftData (@Model, @Query)
Visualization: Swift Charts (native)

Architecture: MVVM with SwiftUI
Pattern: Reactive data binding
State Management: @State, @Environment, @Query
Preview Support: Full SwiftUI previews

QUICK START:
============

1. Requirements:
   - macOS 13.0+ (Ventura)
   - Xcode 15.0+
   - iOS 17 simulator or device

2. Build & Run:
   $ open BPTracker.xcodeproj
   # In Xcode: Select simulator → Cmd+R

3. First Use:
   - Tap + button
   - Enter BP reading (e.g., 120/80)
   - Enter pulse (e.g., 70)
   - Add notes (optional)
   - Tap Save
   - View in Readings tab
