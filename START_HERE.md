## ⚡ Quick Start (2 Minutes)

### For Users - Just Want to Use It?

1. **Open the project:**
   ```bash
   open BPTracker.xcodeproj
   ```

2. **In Xcode:**
   - Select iPhone 15 simulator (or any iPhone)
   - Press **▶ (Play)** or hit `Cmd+R`
   - Wait ~30 seconds for build
   - App launches!

3. **Start tracking:**
   - Tap the **+** button
   - Enter your blood pressure (e.g., 120/80)
   - Enter pulse (e.g., 70)
   - Tap **Save**
   - Done! 🎉

## 📱 What It Does

**BPTracker** helps you:
- 📝 Log blood pressure readings (systolic/diastolic)
- ❤️ Track pulse/heart rate
- 📊 Visualize trends with interactive charts
- 📈 Analyze patterns and averages
- 📤 Export data to CSV
- 🎨 Enjoy a clean, modern interface
- 🔒 Keep data private (local storage only)

Based on American Heart Association (AHA) medical guidelines.

## ⚠️ Important Notes

### Medical Disclaimer
This app is for **informational purposes only**. Not a medical device. Always consult healthcare professionals for medical advice, diagnosis, or treatment.

### Privacy
All data stored **locally on your device**. No cloud sync, no tracking, no data collection. You control your data.

### Code Signing
When building for a physical device, you'll need to sign the app:
1. Select the project in Xcode
2. Go to "Signing & Capabilities"
3. Check "Automatically manage signing"
4. Sign in with your Apple ID
